## [0.2.0] - 2026-05-28

### Added

- Команда `greet` с флагом `--shout` для громкого приветствия.
- Поддержка Python 3.12.

### Changed

- Сообщение об ошибке при пустом `--name` стало понятнее.

### Fixed

- При отсутствии `requirements.txt` утилита больше не падает, а печатает подсказку.
